Технологии, которыми я хорошо владею.

FreeBSD

Первая инсталляция в качестве шлюза в интернет для домашней локальной сети в 1999 году. После глубокого изучения приёмов администрирования, с 2003 года - настольная ОС. Всегда на столе sendmail, apache, mysql, squid. Всегда собираю ядро под конкретные нужды (конфиги ядра). Хорошо изучил содержимое портов, знаю где там что искать, портирую для FreeBSD софт из других клонов UNIX. С другими клонами UNIX работаю очень редко.

C/C++, Perl

Изучал Си по книге Кернигана и Ритчи ещё работая в СКБ НП в 1987-1989 гг. В 90-х, в условиях полуразрушенной постсоветской экономики, работы в этой сфере было мало. Я практиковался, делая студентам курсовые и лабораторные. Писал примитивный биллинг (libpcap), программу мониторинга работы tcp-сервисов ipmon. В BSS - сервис для кластеризации IPDivert, в "Союзтелеком" дополнял и правил SMSC (SMPP v3.4), некоторые сервисы. Десятки тысяч строк.

На C++ опыта поменьше, но также немало. Для ЗАО "Союзтелеком" строил MMS VAS (приём сообщений от MMS-центров сотовых операторов и хранение их в базе данных; был реализован MM7: PAP, EAIF), принимались ММС от МТС и Мегафона.

На Perl пишу с начала использования FreeBSD, сначала для нужд администрирования, затем CGI и пошло-поехало. Есть даже сервер для SMS-игры.

Владею C/C++/Perl API для программирования сетевых соединений, libmysql/mysql++, libxml2/libxml++, др. Для подготовки документации использую doxygen, для храниения кода - CVS. Изучаю DocBook.

MySQL

Всегда использую для любых проектов, как настольную БД. Многие проекты для других БД проектировались и обкатывались на MySQL, если не было непосредственного доступа к рабочей БД заказчика. Устанавливаю, администрирую, программирую внешние API и триггеры на SQL, проектирую. Смогу отличить третью нормальную форму от формы Бойля-Кодда. Методами графического дизайна, UML не владею, не пользуюсь, не приходилось, хотя, конечно, считаю необходимым, читаю Мартина Фаулера. Почти всегда использую только консольные приложения: mysql и mysqldump.

С другими БД работал эпизодически и другие БД не администрировал. В ЗАО "Союзтелеком" работал с Oracle на Си (OCI), Perl.

WEB-технологии: HTML/XML/CSS/XSLT/etc.

Считаю очень перспективным реализацию интерфеса почти для любого сервиса через веб. Это делает использование ресурсов сервера простым и универсальным (браузер стоит у всех), интергированым в глобальное информационное пространство (одинаково может использоваться и на рабочем столе, и в локальной сети, и в интернет), обеспечивающим коммерческую безопасность (SSL) и позволяющим легко масштабировать решение: исходя из потребностей, можно использовать для сервера всё, от старенького P166 MMX, пылящегося в углу, до мощного многопроцессорного кластера.

Редактирую HTML/XML простым текстовым редактором (предпочитаю vi). Стараюсь следить за новыми технологими от W3C. Затачивать HTML под xHTML начал ещё до появления этого стандарта для возможности автоматической обработки. JavaScript, DOM - само собой разумеется.

PHP не владею.
